# CUDA Image Restoration


В данной работе реализована система восстановления изображений после импульсного шума (salt & pepper noise) с использованием CUDA Python.

Программа выполняет:

- загрузку изображения;
- добавление случайного шума;
- восстановление изображения median filter;
- сравнение CPU и GPU реализации;
- вычисление метрик качества;
- визуализацию результатов.

---

# Используемые технологии

- Python
- NumPy
- Numba CUDA
- Matplotlib
- Pillow

---

# Алгоритм работы

## 1. Загрузка изображения

Изображение загружается в оттенках серого:

```python
image = Image.open(path).convert("L")
```

---

## 2. Генерация импульсного шума

К изображению добавляется salt & pepper noise:

- часть пикселей становится чёрной;
- часть пикселей становится белой.

---

## 3. Median Filter

Для восстановления используется медианный фильтр 3×3.

Для каждого пикселя:

1. берётся окно 3×3;
2. значения сортируются;
3. центральный элемент заменяется медианой.
